# Avito QA Test — Инструкция по запуску

## Требования

Перед запуском убедитесь, что у вас установлены:

| Инструмент | Версия | Как проверить |
|------------|--------|---------------|
| **Java** | 11 или 17 | `java -version` |
| **Maven** | 3.6+ | `mvn -v` |
| **Allure** (для отчётов) | 2.20+ | `allure --version` |

> Если `allure` не установлен:
> - **Windows**: `choco install allure`
> - **Mac**: `brew install allure`
> - **Linux**: [скачать вручную](https://docs.qameta.io/allure/#_linux)

### 1. Склонируйте репозиторий

```
git clone https://github.com/max2xam2/Avito_Qa_Test_Oskin.git
```

Или скачайте ZIP-архив и распакуйте его

### 2. Перейдите в корневую папку проекта
```
cd /здесь укажите путь до директории с проектом  
```

### 3. Запустите тесты через терминал 
```
mvn tets
```

### 4. Запустите формирование отчетности с помощью Allure, браузер откроется самсстоятельно
```
mvn allure:serve
```
